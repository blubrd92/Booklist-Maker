Booklister Helper

Preferred branch

Case-insensitive substring of your branch name as it appears in the Availability table (or the branch code, like SR). When set, the extension picks that branch's call number when capturing a book. Leave blank to use BiblioCommons' own "local branch" detection — works automatically when you're signed in or browsing from your library's IP range.

Accumulate mode

When on, each click on a single book record appends the book to a list and copies the entire list to the clipboard. The toolbar badge shows the running count. Useful when you're browsing individual books and want to gather 13-15 of them before pasting into Booklister all at once. Right-click the toolbar icon and choose Clear accumulated list to start over, or use the button below.

List-page captures (clicking the icon on a curated list URL) are independent of this setting — they always copy the entire list to the clipboard fresh.

Save

Booklister Helper captures the book record from the page you're looking at. It does not collect or transmit any data outside your browser; the only network calls it makes are the same Availability- by-location request your library's catalog page already makes when you click that button, plus a fetch of the cover image bytes from your library's cover provider (typically Syndetics) so the cover travels with the booklist.
